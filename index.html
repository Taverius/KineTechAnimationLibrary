    About

    • The project is in it's infancy and is currently a side project for Lead Engineer Dustin Judson(@KineMorto), however the library will continue to grow and become more robust as more animations are needed and the Mod-Community for Kerbal Space Program as a whole makes requests and suggestions.
    • Originally started as a single class of Animation Logic for then Mod-Author Bac9 and quickly and effortlessly developed into an extensive set of animation modules for various purposes through abstraction.

    Features

    • Numerous type of Animation Modules for Parts in Kerbal Space Program
    • Automatic generation of documentation for .cfg files via attributes.
    • DEBUG and RELEASE builds that expose relevant fields directly within the UIPartController.
    • Many more to come!
